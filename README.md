RHrebecek\PHPMail
======

Requirements
------------

RHrebecek\PHPMail requires PHP 5.4 or higher.


Installation
------------

The best way to install RHrebecek\PHPMail is using  [Composer](http://getcomposer.org/):

```sh
$ composer require rhrebecek/php-mail
```
